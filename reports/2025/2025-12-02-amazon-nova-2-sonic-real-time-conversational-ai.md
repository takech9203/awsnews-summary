# Amazon Bedrock - Amazon Nova 2 Sonic 音声対音声モデルの一般提供開始

**リリース日**: 2025 年 12 月 2 日  
**サービス**: Amazon Bedrock  
**機能**: Amazon Nova 2 Sonic


## 概要

AWS は Amazon Nova 2 Sonic の一般提供を発表しました。これは、アプリケーションに自然でリアルタイムの音声会話をもたらす音声対音声 (speech-to-speech) 基盤モデルです。このモデルは、業界をリードする会話品質、価格性能比、そして開発者が音声アプリケーションを構築するための最高クラスの音声理解を提供します。

Nova 2 Sonic は、表現力豊かな音声、自然なターンテイキング、ユーザーの割り込みのシームレスな処理を実現します。人間の好み評価では、リスナーは全体的なリスニング体験において、他の主要モデルよりも一貫して Nova 2 Sonic の出力を好むことが示されています。

**アップデート前の課題**

- 従来の音声 AI は、ユーザーが「何を言ったか」だけでなく「どのように言ったか」という音響コンテキストを保持することが困難だった
- 複数言語をサポートする音声アプリケーションでは、言語ごとに別々のモデルが必要だった
- 電話システムとの統合には複雑な技術要件があり、開発者の負担が大きかった

**アップデート後の改善**

- 音響コンテキストを保持し、ユーザーの発話方法に応じて応答を適応させる真に流暢な音声インタラクションを実現
- ポリグロット音声により、同じ会話内で複数言語を切り替え可能に
- Amazon Connect、Vonage、Twilio、Audiocodes などの主要テレフォニープロバイダーとの直接統合を提供


## サービスアップデートの詳細

### 主要機能

1. **改善された音声理解**
   - 英数字入力、短い発話、8KHz テレフォニー音声入力の精度向上
   - 異なるアクセントやバックグラウンドノイズへの堅牢性向上
   - Big Bench Audio、BFCL、ComplexFuncBench などのベンチマークで他の主要モデルを上回る性能

2. **拡張されたグローバルリーチとポリグロット音声**
   - 英語、フランス語、イタリア語、ドイツ語、スペイン語に加え、ポルトガル語とヒンディー語をサポート
   - 同じ会話内で言語を切り替えられるポリグロット音声
   - コードスイッチング (言語混合) を自然に処理

3. **自然なターンテイキング**
   - 設定可能な音声アクティビティ検出感度 (高、中、低)
   - 高感度: 最速の応答時間に最適化
   - 低感度: ユーザーが考えをまとめる時間を確保

4. **シームレスなクロスモーダルインタラクション**
   - 同じセッション内でテキストと音声入力を切り替え可能
   - モダリティ間でコンテキストを維持
   - IVR アプリケーションのナビゲーションにテキストメタデータを使用可能

5. **高度なマルチエージェント機能**
   - 非同期ツール呼び出しにより、ツール実行中も新しいユーザー入力に応答
   - 複数の無関係なタスクを同時に管理しながらエンゲージメントを維持

6. **強化されたテレフォニーとプラットフォーム統合**
   - Amazon Connect、Vonage、Twilio、Audiocodes との直接統合
   - LiveKit、Pipecat などのメディアプラットフォームをサポート
   - オーディオコーデック最適化、セッションライフサイクル管理を自動処理

## 設定方法

### 前提条件

1. AWS アカウント
2. Amazon Bedrock へのアクセス権限
3. 対応リージョンでの利用

### 手順

#### ステップ 1: モデル ID の指定

Nova 2 Sonic は Amazon Bedrock を通じて利用可能です。モデル ID `amazon.nova-2-sonic-v1:0` を使用します。

#### ステップ 2: 双方向ストリーミング API の使用

```python
import boto3

# Bedrock Runtime クライアントの作成
bedrock_runtime = boto3.client('bedrock-runtime', region_name='us-east-1')

# Nova 2 Sonic を使用した音声対音声の会話
# 双方向ストリーミング API を使用
# 詳細は Amazon Nova Sonic Speech-to-Speech Model Samples を参照
```

既存の Nova Sonic アプリケーションからのアップグレードは、モデル ID を更新するだけで完了します。

## ユースケース

### ユースケース 1: 多言語カスタマーサポート

**シナリオ**: グローバル企業のカスタマーサポートセンターで、複数言語の顧客に対応

**実装例**:
```
ポリグロット音声 (Tiffany) を使用:
- 顧客が英語で会話を開始
- 途中でスペイン語に切り替え
- 同じ音声特性を維持しながらシームレスに対応
```

**効果**: 言語ごとに別々のモデルを用意する必要がなく、一貫した顧客体験を提供

### ユースケース 2: IVR システムの自動化

**シナリオ**: 予約システムへの自動発信と対話

**実装例**:
```
クロスモーダル機能を活用:
- テキストでパーソナライズされた挨拶を指示
- キーパッドトーンを表すテキストメタデータで IVR をナビゲート
- ユーザーに代わって予約を完了またはボイスメールを残す
```

**効果**: 複雑な IVR システムとの対話を自動化

### ユースケース 3: 教育アプリケーション

**シナリオ**: 言語学習や特別なコミュニケーションニーズを持つユーザー向けアプリ

**実装例**:
```
低感度のターンテイキング設定:
- ユーザーが考えをまとめる時間を確保
- 異なるコミュニケーション好みに対応
- 自然なペースでの会話を実現
```

**効果**: ユーザーのペースに合わせた柔軟な会話体験
