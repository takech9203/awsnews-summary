# AWS Payment Cryptography - API 料金最大 63% 削減と階層型キー料金の導入

**リリース日**: 2025 年 12 月 17 日
**サービス**: AWS Payment Cryptography
**機能**: API 料金の大幅削減と階層型キー料金

## 概要

AWS Payment Cryptography が API リクエストの料金を最大 63% 削減し、新しい第 4 階層を導入しました。この変更により、すべてのリージョンで料金が統一され、高ボリュームのワークロードをコストを気にせずサービスに移行できるようになります。

また、定額制から階層型キー料金に移行し、大量の暗号化キーを管理する必要があるお客様をより良くサポートします。新しい料金モデルにより、キー管理と決済処理の両方の運用をコスト効率よくスケールできます。

**アップデート前の課題**

- 高ボリュームワークロードでの料金が懸念事項だった
- リージョン間で料金が異なっていた
- 定額制のキー料金が大量キー管理に不向きだった
- 専用の決済 HSM からの移行コストが高かった

**アップデート後の改善**

- API 料金が最大 63% 削減
- すべてのリージョンで料金が統一
- 階層型キー料金で大量キー管理をサポート
- 高ボリュームワークロードの移行が容易に

## サービスアップデートの詳細

### 主要機能

1. **API 料金の大幅削減**
   - 既存の階層全体で最大 63% の削減
   - 新しい第 4 階層の導入
   - 高ボリューム利用でさらにコスト効率が向上

2. **リージョン間料金の統一**
   - すべての AWS リージョンで同一料金
   - グローバル展開の計画が容易に

3. **階層型キー料金**
   - 定額制から階層型に移行
   - 大量のキー管理をコスト効率よくサポート
   - ボリュームに応じた料金最適化

## 技術仕様

### AWS Payment Cryptography の概要

| 項目 | 詳細 |
|------|------|
| サービスタイプ | マネージド決済暗号化サービス |
| コンプライアンス | PCI PIN、PCI P2PE 準拠 |
| HSM | PCI 認定 HSM を使用 |
| 対象ユーザー | アクワイアラー、決済ファシリテーター、ネットワーク、スイッチ、プロセッサー、銀行 |

### 料金変更の概要

| 項目 | 変更内容 |
|------|----------|
| API 料金削減 | 最大 63% |
| 新規階層 | 第 4 階層を追加 |
| キー料金 | 定額制 → 階層型 |
| リージョン料金 | 統一 |
| 適用日 | 2025 年 12 月 15 日 |

## メリット

### ビジネス面

- **コスト削減**: API 料金最大 63% 削減で運用コストを大幅に削減
- **予測可能な料金**: リージョン間で統一された料金体系
- **スケーラビリティ**: 高ボリュームワークロードでもコスト効率を維持

### 技術面

- **移行の容易さ**: 専用 HSM からの移行コストが低減
- **柔軟なキー管理**: 階層型料金で大量キー管理をサポート
- **グローバル展開**: 統一料金でマルチリージョン展開が容易

## デメリット・制約事項

### 制限事項

- 料金変更は 2025 年 12 月 15 日以降の利用に適用
- 一部のリージョンでは利用不可

### 考慮すべき点

- 既存の料金見積もりの更新が必要
- 階層型料金への移行に伴う請求書の変更

## ユースケース

### ユースケース 1: 大規模決済プロセッサーの移行

**シナリオ**: 専用の決済 HSM を運用している大規模決済プロセッサーが、クラウドへの移行を検討

**効果**: 最大 63% の料金削減により、移行の ROI が大幅に向上。専用 HSM の運用コストと比較して総所有コストを削減

### ユースケース 2: グローバル決済ネットワーク

**シナリオ**: 複数のリージョンで決済処理を行うグローバル企業

**効果**: リージョン間で統一された料金により、グローバル展開の計画と予算策定が容易に

### ユースケース 3: 成長中のフィンテック企業

**シナリオ**: 急成長中のフィンテック企業が、スケーラブルな決済インフラを必要としている

**効果**: 階層型料金により、成長に合わせてコスト効率よくスケール可能

## 料金

新しい料金体系は 2025 年 12 月 15 日から自動的に適用されます。お客様側での操作は不要です。

詳細な料金情報は [AWS Payment Cryptography 料金ページ](https://aws.amazon.com/payment-cryptography/pricing/) を参照してください。

- 米国西部 (オレゴン)

## まとめ

AWS Payment Cryptography の料金が最大 63% 削減され、階層型キー料金が導入されました。この変更により、専用の決済 HSM からの移行や高ボリュームワークロードの運用がより経済的になります。決済処理をクラウドに移行することを検討している組織にとって、この料金改定は大きなメリットとなります。
