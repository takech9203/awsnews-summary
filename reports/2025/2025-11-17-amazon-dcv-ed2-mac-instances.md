# Amazon DCV - EC2 Mac インスタンスサポート

**リリース日**: 2025 年 11 月 17 日  
**サービス**: Amazon DCV  
**機能**: EC2 Mac インスタンスでの高性能リモートデスクトップ


## 概要

Amazon DCV は、Amazon EC2 Mac インスタンスをサポートしました。この機能により、4K 解像度と 60 FPS のパフォーマンスで高性能なリモートデスクトップアクセスが可能になります。

Windows、Linux、macOS、または Web クライアントから接続でき、タイムゾーンリダイレクションやオーディオ出力などの機能も利用できます。

**アップデート前の課題**

- 以前は EC2 Mac インスタンスへのリモートアクセスに VNC などの汎用プロトコルを使用する必要があり、高解像度・高フレームレートでの表示が困難だった
- 以前はリモートデスクトップ接続の品質が低く、グラフィック作業や精密な開発作業に支障があった
- 以前はタイムゾーンリダイレクションやオーディオ出力などの機能を個別に設定する必要があった

**アップデート後の改善**

- 今回のアップデートにより、4K 解像度と 60 FPS の高品質なリモートデスクトップアクセスが可能になった
- 今回のアップデートにより、Windows、Linux、macOS、Web ブラウザなど複数のクライアントから統一された方法で接続できるようになった
- 今回のアップデートにより、タイムゾーンリダイレクションやオーディオ出力などの機能が標準で利用可能になった


## サービスアップデートの詳細

### 主要機能

1. **高性能リモートデスクトップ**
   - 4K 解像度サポート
   - 60 FPS のパフォーマンス
   - 低レイテンシー接続

2. **マルチプラットフォームクライアント**
   - Windows クライアント
   - Linux クライアント
   - macOS クライアント
   - Web ブラウザクライアント

3. **追加機能**
   - タイムゾーンリダイレクション
   - オーディオ出力
   - USB デバイスリダイレクション


## 技術仕様

### サポートされる EC2 Mac インスタンス

| インスタンスタイプ | プロセッサ | 用途 |
|------------------|----------|------|
| mac1.metal | Intel | 一般的な macOS 開発 |
| mac2.metal | Apple M1 | Apple Silicon 開発 |
| mac2-m2.metal | Apple M2 | 最新 Apple Silicon 開発 |
| mac2-m2pro.metal | Apple M2 Pro | 高性能開発 |

### DCV の仕様

| 項目 | 詳細 |
|------|------|
| 最大解像度 | 4K (3840x2160) |
| フレームレート | 最大 60 FPS |
| プロトコル | QUIC, WebSocket |
| 暗号化 | TLS 1.3 |


## 設定方法

### 前提条件

1. EC2 Mac インスタンスが起動済み
2. DCV サーバーがインストール済み
3. セキュリティグループで DCV ポートが開放済み

### 手順

#### ステップ 1: DCV サーバーのインストール

```bash
# macOS 上で DCV サーバーをインストール
brew install --cask nice-dcv-server
```

#### ステップ 2: DCV サーバーの起動

```bash
sudo /usr/local/bin/dcvserver start
```

#### ステップ 3: セッションの作成

```bash
dcv create-session --type virtual my-session
```

#### ステップ 4: クライアントからの接続

DCV クライアントまたは Web ブラウザから接続します。


## メリット

### ビジネス面

- **リモート開発**: どこからでも Mac 開発環境にアクセス
- **コスト効率**: 物理 Mac の購入・管理が不要
- **スケーラビリティ**: 必要に応じてインスタンスを追加

### 技術面

- **高品質**: 4K/60FPS の高品質表示
- **低レイテンシー**: 快適な操作感
- **セキュリティ**: TLS 暗号化による安全な接続


## デメリット・制約事項

### 制限事項

- EC2 Mac インスタンスは専有ホストが必要
- 最小利用期間は 24 時間

### 考慮すべき点

- ネットワーク帯域幅の要件
- クライアント側のハードウェア要件
