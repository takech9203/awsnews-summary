# AWS Network Firewall - AWS パートナーからのマネージドルール

**リリース日**: 2025 年 11 月 19 日  
**サービス**: AWS Network Firewall  
**機能**: AWS パートナーからのマネージドルールサポート


## 概要

AWS Network Firewall は、AWS パートナーからのマネージドルールをサポートしました。この新機能により、AWS パートナーが提供する専門家によってキュレーションされ、自動的に更新されるセキュリティルールを、ネットワークファイアウォールポリシー内に直接デプロイできます。

AWS Network Firewall コンソールから数クリックで事前設定されたルールグループを統合でき、カスタムルールを管理する運用オーバーヘッドなしに包括的な保護を提供します。

**アップデート前の課題**

- 以前は Network Firewall のルールを自前で作成・管理する必要があり、セキュリティ専門知識と運用負担が必要だった
- 以前は新しい脅威に対応するためにルールを手動で更新する必要があった
- 以前は業界専門家のセキュリティルールを簡単に統合する方法がなかった

**アップデート後の改善**

- 今回のアップデートにより、AWS パートナーが提供するマネージドルールを数クリックでデプロイできるようになった
- 今回のアップデートにより、パートナーによって自動的に更新されるルールで最新の脅威に対応できるようになった
- 今回のアップデートにより、カスタムルール管理の運用オーバーヘッドなしに包括的な保護を実現できるようになった


## サービスアップデートの詳細

### 主要機能

1. **マネージドルールの統合**
   - AWS パートナーが提供するルールグループ
   - 自動更新による最新の脅威対策
   - 数クリックでのデプロイ

2. **サブスクリプションオプション**
   - AWS Network Firewall コンソールから購読
   - AWS Marketplace ウェブサイトから購読
   - 統合請求と長期契約による割引

3. **対応パートナー**
   - Check Point
   - Fortinet
   - Infoblox
   - Lumen
   - Rapid7
   - ThreatSTOP
   - Trend Micro


## 技術仕様

### マネージドルールの特徴

| 特徴 | 説明 |
|------|------|
| 自動更新 | パートナーによる継続的な更新 |
| 専門家キュレーション | 業界専門家によるルール設計 |
| 統合デプロイ | コンソールから簡単に統合 |

### 対応パートナー一覧

| パートナー | 専門分野 |
|-----------|---------|
| Check Point | 包括的なセキュリティ |
| Fortinet | ネットワークセキュリティ |
| Infoblox | DNS セキュリティ |
| Lumen | 脅威インテリジェンス |
| Rapid7 | 脆弱性管理 |
| ThreatSTOP | 脅威防御 |
| Trend Micro | クラウドセキュリティ |


## 設定方法

### 前提条件

1. AWS Network Firewall が設定済み
2. AWS Marketplace へのアクセス権限
3. 適切な IAM 権限

### 手順

#### ステップ 1: マネージドルールの選択

AWS Network Firewall コンソールまたは AWS Marketplace からマネージドルールを選択します。

#### ステップ 2: サブスクリプション

選択したマネージドルールにサブスクライブします。

#### ステップ 3: ファイアウォールポリシーへの統合

マネージドルールグループをファイアウォールポリシーに追加します。


## メリット

### ビジネス面

- **運用効率**: カスタムルール管理の負担軽減
- **専門知識の活用**: セキュリティ専門家のルールを利用
- **コスト最適化**: 長期契約による割引

### 技術面

- **自動更新**: 新しい脅威への迅速な対応
- **簡単な統合**: 数クリックでデプロイ
- **包括的な保護**: 複数パートナーのルールを組み合わせ可能


## デメリット・制約事項

### 制限事項

- マネージドルールには追加のサブスクリプション料金が発生
- パートナーによってルールの内容や更新頻度が異なる

### 考慮すべき点

- 組織のセキュリティ要件に合ったパートナーを選択
- 既存のカスタムルールとの整合性を確認


## ユースケース

### ユースケース 1: 包括的な脅威防御

**シナリオ**: 最新の脅威に対する包括的な保護を実装したい

**効果**: 複数のパートナーからのマネージドルールを組み合わせて多層防御を実現

### ユースケース 2: コンプライアンス対応

**シナリオ**: 業界固有のコンプライアンス要件を満たすセキュリティルールが必要

**効果**: 業界に特化したパートナーのルールグループを使用してコンプライアンスを確保

### ユースケース 3: セキュリティ運用の効率化

**シナリオ**: セキュリティチームのルール管理負担を軽減したい

**効果**: マネージドルールの自動更新により、運用オーバーヘッドを削減


## 料金

AWS Network Firewall の標準料金に加えて、マネージドルールのサブスクリプション料金が発生します。

- AWS Marketplace を通じた統合請求
- 長期契約による割引オプション

詳細は各パートナーの AWS Marketplace ページを参照してください。


## 利用可能リージョン

AWS Network Firewall と AWS Marketplace が利用可能なすべての AWS 商用リージョンで利用可能です。
