# AWS Glue - ニュージーランドリージョンで利用可能に

**リリース日**: 2026年01月20日
**サービス**: AWS Glue
**機能**: アジアパシフィック (ニュージーランド) リージョンでの提供開始

## 概要

AWS Glue が、アジアパシフィック (ニュージーランド) リージョンで利用可能になりました。これにより、ニュージーランドの顧客は、ローカルリージョンで AWS Glue を使用してデータ統合、ETL (抽出、変換、ロード) ワークロードを実行できるようになります。

AWS Glue は、分析、機械学習、アプリケーション開発のためのデータの検出、準備、統合を容易にするサーバーレスデータ統合サービスです。Glue を使用すると、複数のソースからデータを検出して接続し、一元化されたデータカタログでデータを管理し、ビジュアルおよびコードベースのインターフェイスを使用して ETL ジョブを作成、実行、監視できます。

**アップデート前の課題**

- AWS Glue がニュージーランドリージョンで利用できなかった
- ニュージーランドのデータを他のリージョンに転送する必要があり、レイテンシーとコストが増加していた
- データレジデンシー要件を満たすのが困難だった

**アップデート後の改善**

- ニュージーランドリージョンで AWS Glue を利用できるようになった
- データをローカルリージョンに保持したまま ETL ワークロードを実行可能になった
- レイテンシーの削減とデータ転送コストの低減を実現

## サービスアップデートの詳細

### AWS Glue の主要機能

- **サーバーレス**: インフラストラクチャ管理が不要で、使用した分だけ課金
- **データカタログ**: メタデータを一元管理し、データの検出を容易化
- **ETL ジョブ**: ビジュアルまたはコードベースで ETL ジョブを作成
- **データ統合**: 複数のデータソースを統合し、分析や機械学習に活用

## メリット

### ビジネス面

- **グローバル展開の拡大**: ニュージーランドで AWS Glue を利用可能
- **データレジデンシー対応**: データをニュージーランド国内に保持したまま ETL を実行
- **コスト削減**: リージョン間のデータ転送コストを削減

### 技術面

- **低レイテンシー**: ローカルリージョンでの ETL 処理により低レイテンシーを実現
- **サーバーレス**: インフラストラクチャ管理の負担を軽減
- **スケーラビリティ**: 必要に応じて自動的にスケーリング

## ユースケース

### ユースケース1: ローカルデータ統合

**シナリオ**: ニュージーランド国内の複数のデータソースを統合し、分析用のデータウェアハウスにロード

**効果**: データをニュージーランドリージョンに保持したまま、低レイテンシーで ETL ワークロードを実行

### ユースケース2: データレジデンシー要件への対応

**シナリオ**: 規制によりデータをニュージーランド国内に保持する必要があるデータパイプライン

**効果**: AWS Glue をニュージーランドリージョンで使用し、データレジデンシー要件を満たす

### ユースケース3: リアルタイムデータ処理

**シナリオ**: ニュージーランドのアプリケーションからリアルタイムにデータを収集し、処理

**効果**: ローカルリージョンでの処理により、低レイテンシーでリアルタイムデータ処理を実現

## 利用可能リージョン

AWS Glue は、アジアパシフィック (ニュージーランド) リージョンを含む多数のリージョンで利用可能です。最新のリージョン情報は [AWS Region Table](https://aws.amazon.com/about-aws/global-infrastructure/regional-product-services/) を参照してください。

## 関連サービス・機能

- **AWS Glue Data Catalog**: メタデータの一元管理
- **AWS Glue Studio**: ビジュアル ETL ジョブ作成ツール
- **AWS Glue DataBrew**: ノーコードデータ準備ツール
- **Amazon Athena**: S3 上のデータを SQL で直接クエリ

AWS Glue のニュージーランドリージョンでの提供開始により、ニュージーランドの顧客もローカルリージョンでサーバーレスデータ統合と ETL ワークロードを実行できるようになりました。データレジデンシー要件を満たしながら、低レイテンシーでデータパイプラインを構築できます。
