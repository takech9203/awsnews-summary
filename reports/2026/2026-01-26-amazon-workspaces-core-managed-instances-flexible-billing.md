# Amazon WorkSpaces Core - 柔軟な課金オプションの追加

**リリース日**: 2026年01月26日
**サービス**: Amazon WorkSpaces Core
**機能**: マネージドインスタンスの月次料金オプション

## 概要

AWS は、Amazon WorkSpaces Core マネージドインスタンスに柔軟な課金オプションを発表しました。既存の時間単位課金に加えて、月次定額料金が新たに利用可能になります。これにより、エンドユーザーの利用パターンに基づいて最適な料金モデルを選択できるようになります。月次課金は予測可能なフルタイムデスクトップに最適であり、時間単位課金は変動する利用パターンに適しています。どちらのオプションも従量課金制で、長期契約は不要です。

Amazon WorkSpaces Core マネージドインスタンスは、高度にカスタマイズ可能なインスタンス構成で仮想デスクトップインフラストラクチャ (VDI) の移行を簡素化します。WorkSpaces Core マネージドインスタンスは、顧客の AWS アカウント内にリソースをプロビジョニングし、永続的ワークロードと非永続的ワークロードの両方のインフラストラクチャライフサイクル管理を処理します。

柔軟な課金により、顧客は永続的なデスクトップワークロードに対して予測可能なコストを実現し、同一デプロイメント内で時間単位と月次課金を混在させる柔軟性を得られます。Citrix、Workspot、Dizzion、Leostream などの VDI パートナーは、新しい WorkSpaces API 課金機能と統合することで、インスタンス作成時に月次課金オプションを有効にできます。

**アップデート前の課題**

- 時間単位課金のみが利用可能で、常時稼働するデスクトップでは課金が積み上がりコストが不明確だった
- フルタイムユーザーに対して最適化された料金プランが存在しなかった
- WorkSpaces Core と EC2 の課金が分離されており、請求書が複雑だった

**アップデート後の改善**

- 常時利用のデスクトップに対して月次定額料金を選択できるようになり、コストの予測が容易になった
- 時間単位と月次課金を同一デプロイメント内で混在させることが可能になった
- 時間単位課金が Amazon WorkSpaces で統合請求されるようになり、請求が簡素化された

## サービスアップデートの詳細

### 主要機能

1. **月次定額料金オプション**
   - 常時稼働のデスクトップワークロードに最適化された月次固定料金
   - 実際の VDI ユースケースに最適化され、常時稼働時は時間単位課金と比較してコスト削減を実現
   - 従量課金制で長期契約不要

2. **柔軟な課金モデルの選択**
   - 時間単位課金: 変動する利用パターンに適している
   - 月次課金: 予測可能なフルタイムデスクトップに適している
   - 同一デプロイメント内で両方の課金モデルを混在可能

3. **統合請求の導入**
   - 時間単位課金のユーティリティレートが Amazon WorkSpaces で一括請求されるように変更
   - 以前は EC2 と WorkSpaces で分割請求されていた
   - オンデマンド時間単位利用の実効レートに変更なし

4. **VDI パートナー統合**
   - Citrix、Workspot、Dizzion、Leostream が新しい WorkSpaces API 課金機能と統合可能
   - インスタンス作成時に月次課金オプションを有効化できる
   - 時間単位課金がデフォルトの課金オプションとして維持される

## 技術仕様

### 課金オプション

| 課金モデル | 特徴 | 最適な利用パターン |
|-----------|------|-------------------|
| 月次定額料金 | 固定月額料金、無制限利用 | フルタイムデスクトップ、常時稼働ワークロード |
| 時間単位課金 | 利用時間に応じた課金 | パートタイム、変動利用パターン |

### API 変更履歴

このアップデートでは、WorkSpaces Core API に新しい課金機能が追加されました。VDI パートナーはインスタンス作成時に課金オプションを指定できます。

## 設定方法

### 前提条件

1. Amazon WorkSpaces Core マネージドインスタンスへのアクセス権限
2. VDI 管理ソフトウェアライセンス (サードパーティまたは AWS Marketplace から取得)
3. VDI パートナーソリューション (Citrix、Workspot、Dizzion、Leostream など) との統合

### 手順

#### ステップ1: 課金モデルの選択

利用パターンに基づいて適切な課金モデルを選択します。

- **月次課金**: フルタイムユーザー、常時稼働デスクトップ
- **時間単位課金**: パートタイムユーザー、断続的な利用

#### ステップ2: インスタンス作成時の課金オプション指定

VDI パートナーソリューションを使用して、WorkSpaces Core API 経由でインスタンスを作成する際に課金オプションを指定します。

#### ステップ3: 課金モデルの混在

同一デプロイメント内で、ユーザーや利用パターンに応じて時間単位と月次課金を混在させることが可能です。

## メリット

### ビジネス面

- **コストの予測可能性**: 月次定額料金により、常時稼働デスクトップのコストを正確に予測できる
- **コスト最適化**: 利用パターンに応じて最適な課金モデルを選択することで、全体的なコストを削減できる
- **請求の簡素化**: 統合請求により、請求書の管理が容易になる

### 技術面

- **柔軟性**: 同一デプロイメント内で異なる課金モデルを混在させることができる
- **VDI パートナー統合**: 既存の VDI 管理ソフトウェアと統合して課金オプションを制御可能
- **インフラストラクチャ管理の簡素化**: WorkSpaces Core がインフラストラクチャのライフサイクル管理を処理

## デメリット・制約事項

### 制限事項

- 月次課金と時間単位課金の切り替えには、インスタンスの再作成が必要になる可能性がある
- VDI パートナーソリューションとの統合が必要
- 時間単位課金がデフォルトの課金オプションとして設定されている

### 考慮すべき点

- 利用パターンを正確に把握し、適切な課金モデルを選択する必要がある
- サードパーティ VDI 管理ソフトウェアのライセンスコストが別途必要
- 追加の OS やソフトウェアのコストも考慮する必要がある

## ユースケース

### ユースケース1: フルタイムデスクトップワーカー

**シナリオ**: 100 人のフルタイムユーザーが毎日 8 時間以上デスクトップを使用する一般的な生産性業務

**実装例**:
- Value Bundle (1vCPU、2GB メモリ、80GB ルートボリューム、10GB ユーザーボリューム) を選択
- 月次定額料金オプションを選択
- 月額料金: $16.80 / 仮想デスクトップ
- 月間総コスト: $1,680

**効果**: 時間単位課金と比較して、常時稼働時のコストを削減し、予測可能な月額コストを実現

### ユースケース2: パートタイムユーザー

**シナリオ**: 10 人のパートタイムユーザーが週 5 時間 (月 20 時間) のみ高性能マシンを使用

**実装例**:
- Power Bundle (4vCPU、16GB メモリ、80GB ルートボリューム、10GB ユーザーボリューム) を選択
- 時間単位課金オプションを選択
- 月額固定費: $5.80 / 仮想デスクトップ
- 時間単位料金: $0.51 / 時間
- 月間総コスト: $58 (アクセスコスト) + $102 (利用コスト) = $160

**効果**: 利用時間が限定的な場合、時間単位課金により無駄なコストを削減

### ユースケース3: ハイブリッド環境

**シナリオ**: 組織内にフルタイムユーザーとパートタイムユーザーが混在

**実装例**:
- フルタイムユーザー: 月次定額料金を選択
- パートタイムユーザー: 時間単位課金を選択
- 同一デプロイメント内で両方の課金モデルを混在

**効果**: 各ユーザーの利用パターンに最適化された課金モデルを適用し、全体的なコストを最適化

## 料金

料金は選択するバンドル (CPU、メモリ、ストレージ構成) と課金モデルによって異なります。

### 料金例 (US West Oregon リージョン)

| バンドル | 月次定額料金 | 時間単位課金 (固定費 + 時間料金) |
|---------|-------------|------------------------------|
| Value (1vCPU, 2GB) | $16.80/月 | $5.80/月 + $0.51/時間 |
| Power (4vCPU, 16GB) | 価格はドキュメント参照 | $5.80/月 + $0.51/時間 |

詳細な料金情報は [WorkSpaces for VDI partners pricing page](https://aws.amazon.com/workspaces/vdi-partners/pricing/) を参照してください。
