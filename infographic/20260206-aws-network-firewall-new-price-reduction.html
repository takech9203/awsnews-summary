概要

AWS Network Firewall で 2 つの料金改善が発表されました。 まず、セカンダリエンドポイントにも NAT Gateway 割引が適用されるようになりました。 また、TLS インスペクションを有効にする Advanced Inspection の追加データ処理料金が 13 リージョンで撤廃されました。

アップデート前

  • NAT Gateway 割引はプライマリエンドポイントのみ
  • Advanced Inspection に $0.001〜$0.009/GB の追加料金
  • 複数 VPC 構成でコストが増大

アップデート後

  • NAT Gateway 割引がセカンダリにも適用
  • 13 リージョンで TLS インスペクション追加料金撤廃
  • 自動適用、顧客側でのアクション不要
